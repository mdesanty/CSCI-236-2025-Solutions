Should I use source control?

Here is a comparison of using source control and not using source control.

Using source control

  • All files safe and sound in the cloud.
  • Can revert to previous versions of files.
  • Can work with others.
  • Can recover from mistakes.

Not using source control

  • Files only exist on local disks.
  • Cannot revert to previous versions of files.
  • More difficult to work with others.
  • More difficult to recover from mistakes.

As you can see, using source control is much better than not using source control.
